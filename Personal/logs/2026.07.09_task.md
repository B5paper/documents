* [v] `g@`操作符

    g@ 是 Vim 内置的操作符，专门用于在 Visual 模式下调用函数，且不会退出 Visual 模式（因为它的设计就是操作符，接受一个 motion 或 Visual 选区）。

    ```vim
    " 映射为操作符
    nnoremap <leader>fn :set opfunc=FindNextSharp<CR>g@
    vnoremap <leader>fn :<C-u>call FindNextSharp(visualmode())<CR>
    ```
