* [v] 

    强制 Vim 在 Screen 环境下使用绝对兼容的重绘序列。请在 ~/.vimrc 中加入以下代码：

    " 检测是否在 screen 或 tmux 环境下
    if &term =~ 'screen' || &term =~ 'tmux'
        " 强制 Vim 使用标准的 VT100/Xterm 清屏和重绘序列
        set t_ti=7[r8
        set t_te=7[r8
        
        " 告诉 Vim 每次重绘时，不仅重绘改变的部分，而是刷新整个终端缓存
        set t_ut=
        
        " 强制同步 Vim 和 Screen 的行/列控制
        set t_cl=Line-cl
    endif

    注：代码中的  是 Esc 键字符。在 Vim 中输入时，可以不用手打，直接复制上面这段代码即可。
