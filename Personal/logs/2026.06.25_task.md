* [v] Python 专用补全

    在 `~/.vimrc` 中添加：

    ```vim
    " 启用文件类型检测和插件
    filetype plugin indent on
    syntax on

    " 设置 Python 的 omnifunc
    autocmd FileType python set omnifunc=python3complete#Complete
    ```
