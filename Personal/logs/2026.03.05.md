* 简述 kmp 算法

    KMP（Knuth-Morris-Pratt）算法是一种高效的**字符串匹配算法**，用于在一个文本串中查找模式串的出现位置。

    **核心思想**

    - **利用已匹配信息**：当匹配失败时，不是从头开始重新匹配，而是利用部分匹配表（next数组）跳过不必要的比较

    - **主串指针不回溯**：主串的指针一直向前移动，只有模式串指针回溯

    **关键概念：部分匹配表（next数组）**

    - 记录模式串中每个位置之前的子串的**最长相等前后缀长度**
    - 前缀：除最后一个字符外的所有头部子串
    - 后缀：除第一个字符外的所有尾部子串

    **算法步骤**

    1. **预处理**：计算模式串的next数组

    2. **匹配过程**：

       - 同时遍历主串和模式串
       - 字符匹配成功：继续比较下一个字符
       - 字符匹配失败：根据 next 数组移动模式串，主串指针不变

    **时间复杂度**

    - 预处理：O(m)，m为模式串长度
    - 匹配过程：O(n)，n为主串长度
    - **总体：O(m+n)**，优于暴力匹配的O(m×n)

    **示例**

    ```
    主串：    ABCABD ABC...
    模式串：  ABCABF
    匹配到F失败时，利用 next 数组，直接移动到：
    主串：    ABCABD ABC...
    模式串：     ABCABF
    ```

    KMP 通过空间换时间的思想，用 next 数组记录已匹配信息，避免了重复比较，是字符串匹配的经典算法。

* 展开来详细讲讲，给出算法细节和实际例子，以及c++代码实现

    KMP 算法详解

    一、算法背景与核心思想

    1.1 为什么需要KMP？

    传统的暴力匹配在失配时，主串指针回溯，模式串从头开始，导致大量重复比较。例如：
    
    ```
    主串：AAAAAAAAAAAAAAAB
    模式串：AAAAB
    ```

    暴力匹配每次在 B 失配后，都要从头开始，效率极低。

    1.2 KMP的核心优化

    - **主串永不回溯**：i指针一直向前移动

    - **模式串智能移动**：利用已匹配信息，跳过不可能成功的比较

    - **关键发现**：当匹配到某个字符失败时，前面已匹配的部分有相同的前缀和后缀，可以直接将前缀移动到后缀的位置

    二、部分匹配表（next 数组）详解

    2.1 基本概念

    **前缀**：除最后一个字符外，字符串的所有头部子串
    **后缀**：除第一个字符外，字符串的所有尾部子串
    **部分匹配值**：前缀和后缀的最长相等长度

    2.2 手工计算 next 数组

    以模式串"ABABC"为例：

    1. **位置0(A)**：无前缀后缀 → next[0] = 0
    2. **位置1(AB)**：前缀{A}，后缀{B}，无相等 → next[1] = 0
    3. **位置2(ABA)**：前缀{A,AB}，后缀{A,BA}，相等{A}长度1 → next[2] = 1
    4. **位置3(ABAB)**：前缀{A,AB,ABA}，后缀{B,AB,BAB}，相等{AB}长度2 → next[3] = 2
    5. **位置4(ABABC)**：前缀{A,AB,ABA,ABAB}，后缀{C,BC,ABC,BABC}，无相等 → next[4] = 0

    最终 next 数组：[0,0,1,2,0]

    2.3 next 数组的优化（nextval 数组）

    原始 next 数组在某些情况下仍有优化空间，比如连续相同字符：

    模式串"AAAAB"：

    - 原始 next：[0,1,2,3,0]
    - 优化后 nextval：[0,0,0,0,0]

    优化原理：如果 `P[i] == P[next[i]]`，那么当i失配时，跳到 `next[i]` 后依然会失配，所以需要递归向前优化。

    三、匹配过程详解

    3.1 匹配规则

    - 当 `P[j] == T[i]` 时：i++, j++
    - 当 `j == 0` 且 `P[j] != T[i]` 时：i++（第一个字符就失配）
    - 当 `P[j] != T[i]` 时： `j = next[j-1]`（模式串右移）

    3.2 完整示例

    主串T：ABABABC
    模式串P：ABABC
    next数组：[0,0,1,2,0]

    **逐步匹配过程：**

    1. i=0,j=0：A==A → i=1,j=1
    2. i=1,j=1：B==B → i=2,j=2
    3. i=2,j=2：A==A → i=3,j=3
    4. i=3,j=3：B==B → i=4,j=4
    5. i=4,j=4：A!=C → j=next[3]=2

    ```
    T: A B A B A B C
    P: A B A B C
            ↑ 失配，j跳到2
    ```

    6. i=4,j=2：A==A → i=5,j=3
    7. i=5,j=3：B==B → i=6,j=4
    8. i=6,j=4：C==C → 匹配成功！

    四、C++代码实现

    ```cpp
    #include <iostream>
    #include <vector>
    #include <string>
    using namespace std;

    class KMP {
    private:
        // 计算next数组（原始版本）
        vector<int> getNext(const string& pattern) {
            int m = pattern.length();
            vector<int> next(m, 0);
            
            for (int i = 1; i < m; i++) {
                int j = next[i - 1];
                
                // 当前字符不匹配时，通过 next 回溯
                while (j > 0 && pattern[i] != pattern[j]) {
                    j = next[j - 1];
                }
                
                // 如果匹配，长度加 1
                if (pattern[i] == pattern[j]) {
                    j++;
                }
                
                next[i] = j;
            }
            
            return next;
        }
        
        // 优化版本的next数组（nextval）
        vector<int> getNextVal(const string& pattern) {
            vector<int> next = getNext(pattern);
            vector<int> nextval = next;
            
            for (int i = 1; i < pattern.length(); i++) {
                // 如果当前字符和要跳转到的字符相同，继续向前优化
                if (pattern[i] == pattern[next[i]]) {
                    nextval[i] = nextval[next[i]];
                }
            }
            
            return nextval;
        }

    public:
        // 查找第一个匹配位置
        int searchFirst(const string& text, const string& pattern) {
            if (pattern.empty()) return 0;
            if (text.empty()) return -1;
            
            vector<int> next = getNext(pattern);
            int n = text.length(), m = pattern.length();
            int j = 0;  // 模式串指针
            
            for (int i = 0; i < n; i++) {
                // 当前字符不匹配时，通过next数组调整j
                while (j > 0 && text[i] != pattern[j]) {
                    j = next[j - 1];
                }
                
                if (text[i] == pattern[j]) {
                    j++;
                }
                
                // 找到完整匹配
                if (j == m) {
                    return i - m + 1;
                }
            }
            
            return -1;
        }
        
        // 查找所有匹配位置
        vector<int> searchAll(const string& text, const string& pattern) {
            vector<int> positions;
            if (pattern.empty() || text.empty()) return positions;
            
            vector<int> next = getNext(pattern);
            int n = text.length(), m = pattern.length();
            int j = 0;
            
            for (int i = 0; i < n; i++) {
                while (j > 0 && text[i] != pattern[j]) {
                    j = next[j - 1];
                }
                
                if (text[i] == pattern[j]) {
                    j++;
                }
                
                if (j == m) {
                    positions.push_back(i - m + 1);
                    j = next[j - 1];  // 继续寻找下一个匹配
                }
            }
            
            return positions;
        }
        
        // 打印匹配过程（调试用）
        void printMatchProcess(const string& text, const string& pattern) {
            cout << "Text:    " << text << endl;
            cout << "Pattern: " << pattern << endl;
            cout << "Next数组: ";
            vector<int> next = getNext(pattern);
            for (int val : next) cout << val << " ";
            cout << endl << endl;
            
            int n = text.length(), m = pattern.length();
            int j = 0;
            int step = 1;
            
            for (int i = 0; i < n; i++) {
                cout << "步骤" << step++ << ": i=" << i << ", j=" << j << ", 比较 " 
                     << text[i] << " 和 " << pattern[j] << endl;
                
                while (j > 0 && text[i] != pattern[j]) {
                    cout << "  失配，j从" << j << "跳到" << next[j-1] << endl;
                    j = next[j - 1];
                }
                
                if (text[i] == pattern[j]) {
                    cout << "  匹配成功，继续" << endl;
                    j++;
                } else {
                    cout << "  第一个字符就失配，i++" << endl;
                }
                
                if (j == m) {
                    cout << "*** 找到匹配，起始位置: " << i - m + 1 << " ***" << endl;
                    j = next[j - 1];
                }
                
                cout << endl;
            }
        }
    };

    // 测试代码
    int main() {
        KMP kmp;
        
        // 测试案例1
        string text1 = "ABABABC";
        string pattern1 = "ABABC";
        
        cout << "=== 测试1: 基本匹配 ===" << endl;
        int pos = kmp.searchFirst(text1, pattern1);
        cout << "第一次匹配位置: " << pos << endl << endl;
        
        // 测试案例2：多个匹配
        string text2 = "ABCABCABC";
        string pattern2 = "ABC";
        
        cout << "=== 测试2: 多个匹配 ===" << endl;
        vector<int> positions = kmp.searchAll(text2, pattern2);
        cout << "所有匹配位置: ";
        for (int p : positions) cout << p << " ";
        cout << endl << endl;
        
        // 测试案例3：无匹配
        string text3 = "ABCDEFG";
        string pattern3 = "XYZ";
        
        cout << "=== 测试3: 无匹配 ===" << endl;
        pos = kmp.searchFirst(text3, pattern3);
        cout << "匹配结果: " << (pos == -1 ? "未找到" : "找到") << endl << endl;
        
        // 测试案例4：连续相同字符
        string text4 = "AAAAAAAAAB";
        string pattern4 = "AAAAB";
        
        cout << "=== 测试4: 连续相同字符 ===" << endl;
        kmp.printMatchProcess(text4, pattern4);
        
        // 测试案例5：空字符串
        cout << "=== 测试5: 边界情况 ===" << endl;
        cout << "空模式串: " << kmp.searchFirst(text1, "") << endl;
        cout << "空主串: " << kmp.searchFirst("", pattern1) << endl;
        
        return 0;
    }
    ```

    五、算法复杂度分析

    5.1 时间复杂度

    - **预处理阶段**：O(m)，每个字符最多比较常数次
    - **匹配阶段**：O(n)，主串每个字符最多比较一次
    - **总体**：O(m + n)

    5.2 空间复杂度

    - O(m)，存储 next 数组

    5.3 与暴力匹配对比

    | 算法 | 最好情况 | 最坏情况 | 平均情况 |
    |------|----------|----------|----------|
    | 暴力匹配 | O(n) | O(m×n) | O(m×n) |
    | KMP | O(n) | O(m+n) | O(m+n) |

    六、KMP的优缺点

    优点

    1. **主串不回溯**：适合流式数据处理
    2. **稳定高效**：时间复杂度始终是线性的
    3. **无需回看**：适合无法回退的输入流

    缺点

    1. **需要额外空间**：存储 next 数组
    2. **预处理开销**：对小规模文本可能得不偿失
    3. **实现复杂**：next 数组的理解和计算较难

    七、实际应用场景

    1. **文本编辑器**：查找替换功能
    2. **搜索引擎**：关键词匹配
    3. **生物信息学**：DNA 序列匹配
    4. **入侵检测**：特征码匹配
    5. **文件比对**：差异比较算法的基础

    KMP 算法通过精巧的预处理，将字符串匹配问题转化为线性时间复杂度的算法，是算法设计思想"空间换时间"的经典范例。

* 详细讲讲 next 数组是怎么构造的，你前面说 next 数组是用前缀和后缀，但是在实际计算的时候，我看到你使用的是 pattern 字符串与自身比较

    **Next数组构造详解**

    你的观察很敏锐！确实，next数组的构造本质上就是**模式串与自身的匹配过程**。让我详细解释这个精妙的设计。

    一、从概念到实现：为什么是自身比较？

    1.1 概念定义回顾

    next[i] 表示：在模式串的 [0...i] 这个子串中，**最长相等前后缀的长度**。

    例如 pattern = "ABABC"：

    - i=2（子串"ABA"）：前缀{A,AB}，后缀{A,BA}，最长相等是"A"长度1
    - i=3（子串"ABAB"）：前缀{A,AB,ABA}，后缀{B,AB,BAB}，最长相等是"AB"长度2

    1.2 关键转化

    求最长相等前后缀，**等价于**：用模式串的前缀去匹配自己的后缀！

    以"ABAB"为例：

    ```
    模式串：A B A B
    后缀：    A B   （从第2个位置开始的"AB"）
    前缀：A B       （开头的"AB"）
    ```

    这就像是用模式串本身（作为模式）去匹配模式串的后缀部分（作为文本）。

    二、手工计算过程详解

    2.1 基本方法

    以 "ABABC" 为例，逐步计算：

    **第 0 步：i=0，子串"A"**

    - 没有前缀和后缀
    - next[0] = 0

    **第1步：i=1，子串"AB"**

    - 比较前缀"A"和后缀"B"：不相等
    - next[1] = 0

    **第2步：i=2，子串"ABA"**

    - 比较长度1：前缀"A" vs 后缀"A" → 相等
    - 比较长度2：前缀"AB" vs 后缀"BA" → 不相等
    - 最大相等长度 = 1
    - next[2] = 1

    **第3步：i=3，子串"ABAB"**

    - 比较长度1：前缀"A" vs 后缀"B" → 不相等
    - 比较长度2：前缀"AB" vs 后缀"AB" → 相等
    - 比较长度3：前缀"ABA" vs 后缀"BAB" → 不相等
    - 最大相等长度 = 2
    - next[3] = 2

    **第4步：i=4，子串"ABABC"**

    - 比较长度1：前缀"A" vs 后缀"C" → 不相等
    - 比较长度2：前缀"AB" vs 后缀"BC" → 不相等
    - 比较长度3：前缀"ABA" vs 后缀"ABC" → 不相等
    - 比较长度4：前缀"ABAB" vs 后缀"BABC" → 不相等
    - 最大相等长度 = 0
    - next[4] = 0

    最终结果：[0, 0, 1, 2, 0]

    三、递推构造方法（核心算法）

    3.1 递推思想

    已知next[0...i-1]，求next[i]。关键观察：

    - next[i] 最多比 next[i-1] 大1
    - 如果 pattern[i] == pattern[next[i-1]]，则 next[i] = next[i-1] + 1
    - 否则，需要回溯寻找更短的前缀

    3.2 递推过程图解

    以 "ABABAC" 为例：

    **初始化：next[0] = 0**

    **i=1，字符'B'**：

    ```
    已知next[0]=0，j=next[0]=0
    比较 pattern[1]='B' 和 pattern[0]='A'：不相等
    j已经为0，无法继续回溯
    next[1]=0
    ```

    **i=2，字符'A'**：

    ```
    j = next[1] = 0
    比较 pattern[2]='A' 和 pattern[0]='A'：相等
    next[2] = j + 1 = 1
    此时j=1，表示已匹配长度为1
    ```

    **i=3，字符'B'**：

    ```
    j = next[2] = 1
    比较 pattern[3]='B' 和 pattern[1]='B'：相等
    next[3] = j + 1 = 2
    此时j=2
    ```

    **i=4，字符'A'**：

    ```
    j = next[3] = 2
    比较 pattern[4]='A' 和 pattern[2]='A'：相等
    next[4] = j + 1 = 3
    此时j=3
    ```

    **i=5，字符'C'**：

    ```
    j = next[4] = 3
    比较 pattern[5]='C' 和 pattern[3]='B'：不相等
    j = next[j-1] = next[2] = 2
    比较 pattern[5]='C' 和 pattern[2]='A'：不相等
    j = next[1] = 1
    比较 pattern[5]='C' 和 pattern[1]='B'：不相等
    j = next[0] = 0
    比较 pattern[5]='C' 和 pattern[0]='A'：不相等
    j=0，无法继续
    next[5]=0
    ```

    四、代码实现细节

    ```cpp
    #include <iostream>
    #include <vector>
    #include <string>
    using namespace std;

    class NextArrayBuilder {
    private:
        // 方法1：基础版本，易于理解
        vector<int> buildNextBasic(const string& pattern) {
            int m = pattern.length();
            vector<int> next(m, 0);
            
            for (int i = 1; i < m; i++) {
                // j表示当前已匹配的前缀长度，也是要比较的前缀位置
                int j = next[i - 1];
                
                // 回溯：当不匹配时，找更短的前缀
                while (j > 0 && pattern[i] != pattern[j]) {
                    cout << "    回溯: j从" << j << "变为" << next[j-1] << endl;
                    j = next[j - 1];
                }
                
                // 如果匹配成功，长度+1
                if (pattern[i] == pattern[j]) {
                    j++;
                    cout << "  匹配成功: next[" << i << "] = " << j << endl;
                } else {
                    cout << "  无匹配: next[" << i << "] = 0" << endl;
                }
                
                next[i] = j;
            }
            
            return next;
        }
        
        // 方法2：双指针版本，更清晰地展示自匹配过程
        vector<int> buildNextTwoPointer(const string& pattern) {
            int m = pattern.length();
            vector<int> next(m, 0);
            
            int j = 0;  // 前缀指针（同时也是已匹配长度）
            
            // i从1开始，是后缀指针
            for (int i = 1; i < m; i++) {
                cout << "比较 suffix[" << i << "]=" << pattern[i] 
                     << " 和 prefix[" << j << "]=" << pattern[j] << endl;
                
                // 不匹配时，回溯前缀指针
                while (j > 0 && pattern[i] != pattern[j]) {
                    cout << "  不匹配，j从" << j << "回溯到" << next[j-1] << endl;
                    j = next[j - 1];
                }
                
                // 匹配成功，前缀指针前移
                if (pattern[i] == pattern[j]) {
                    j++;
                    cout << "  匹配成功，j增加到" << j << endl;
                }
                
                next[i] = j;
                cout << "  next[" << i << "] = " << j << endl << endl;
            }
            
            return next;
        }
        
    public:
        // 打印详细构造过程
        void printBuildProcess(const string& pattern) {
            cout << "模式串: " << pattern << endl;
            cout << "长度: " << pattern.length() << endl;
            cout << "开始构造next数组..." << endl;
            cout << "======================" << endl;
            
            vector<int> next = buildNextTwoPointer(pattern);
            
            cout << "最终next数组: ";
            for (int val : next) cout << val << " ";
            cout << endl;
        }
        
        // 可视化next数组含义
        void visualizeNext(const string& pattern) {
            int m = pattern.length();
            vector<int> next = buildNextBasic(pattern);
            
            cout << "\n=== Next数组可视化 ===" << endl;
            for (int i = 0; i < m; i++) {
                cout << "i=" << i << ", 字符=" << pattern[i] 
                     << ", next[" << i << "]=" << next[i] << " → ";
                
                if (next[i] == 0) {
                    cout << "无相等前后缀" << endl;
                } else {
    ```

    五、递推原理的深入理解

    5.1 为什么可以用递推？

    关键性质：**next[i] 最多比 next[i-1] 大1**

    证明：

    - 假设 next[i-1] = k，表示 pattern[0...k-1] = pattern[i-k...i-1]
    - 当考虑位置i时，如果 pattern[k] == pattern[i]
    - 那么 pattern[0...k] = pattern[i-k...i]
    - 所以 next[i] 至少可以是 k+1
    - 不可能大于 k+1，因为那会要求更长的前缀等于后缀

    5.2 回溯过程的理解

    当 pattern[i] != pattern[j] 时：

    - j 是当前尝试匹配的前缀长度
    - 既然 pattern[0...j-1] 已经匹配了后缀，但下一个字符不匹配
    - 那么我们需要找一个更短的、仍然匹配的前缀
    - 这个更短的前缀长度就是 next[j-1]

      - 因为前2个字符("AB")已经匹配过了

    ```python
            pos = (self.x, self.y)

                self.direction = (self.direction + 1) % 4  # 右转
                self.grid[pos] = 1  # 变黑

                self.grid[pos] = 0  # 变白

            self.x += moves[self.direction][0]
            self.y += moves[self.direction][1]
    ```

    ```cpp
    #include <unistd.h>
    #include <iostream>
    #include <string>
    #include <vector>
    #include <stdio.h>
    #include <algorithm>
    using namespace std;

    void print_vec(vector<int> &vec) {
        for (int i = 0; i < vec.size(); ++i) {
            printf("%d, ", vec[i]);
        }
        putchar('\n');
    }

    vector<int> get_next_arr(const string &pattern) {
        vector<int> next(pattern.size(), 0);
        // pos = 0 时，'a' 没有前缀和后缀，所以 next[0] = 0
        // sub_str_end_pos -> 子串结束位的索引，同时也是 next[] 当前要计算的位的索引
        for (int sub_str_end_pos = 1; sub_str_end_pos < pattern.size(); ++sub_str_end_pos) {
            // 1. 将 'ababc' 拆分成 a, ab, aba, abab, ababc 子串
            string sub_str = pattern.substr(0, sub_str_end_pos + 1);
            for (int prefix_len = 1; prefix_len < sub_str.size(); ++prefix_len) {
                // 2. 对于每个子串，拆分出前缀和后缀
                // 比如 abab，前缀分别为 a, ab, aba，后缀分别为 b, ab, bab
                int subfix_start_pos = sub_str.size() - prefix_len;
                // 3. 线性比较前缀和后缀
                bool prefix_subfix_equal = true;
                for (int p = 0; p < prefix_len; ++p) {
                    if (pattern[p] != pattern[subfix_start_pos+p]) {
                        prefix_subfix_equal = false;
                        break;
                    }
                }
                // 4. 如果前缀和后缀相同，那么更新最大长度
                // 注意，因为 prefix_len 是递增的，所以这里只需要更新就可以了，
                // 不需要 next[sub_str_end_pos] = max(next[sub_str_end_pos], prefix_len);
                if (prefix_subfix_equal) {
                    next[sub_str_end_pos] = prefix_len;
                }
            }
        }
        return next;
    }
    ```
