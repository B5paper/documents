* 在使用 gcc 编译时，如果有这样的编译命令：

    ```bash
    main: main.c A.o B.o
        gcc main.c A.o B.o -o main

    A.o:
        gcc -c A.c -o A.o

    B.o:
        gcc -c B.c -o B.o
    ```

    其中`B.o`中的函数依赖`A.o`中的函数，那么交换`gcc main.c A.o B.o -o main`中`A.o`和`B.o`的顺序，不影响`main`的编译。
