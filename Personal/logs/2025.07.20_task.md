* [v] 调研`#define ASSERT(x) (void)(x)`中 void 的作用

    feedback:

    1. 调研`Coverity`

    1. 调研`Clang Static Analyzer`

    1. `(void)`主要用于防止编译器给出 unused variable 的 warning。

    1. 这个宏的作用有可能是

        ```cpp
        #ifdef DEBUG
            #define ASSERT(x) assert(x)  // 调试阶段启用真实断言
        #else
            #define ASSERT(x) (void)(x) // 发布阶段无害化
        #endif
        ```

        总之是占位符性质。
