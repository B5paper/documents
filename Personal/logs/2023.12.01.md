* c++ 17 初始化一个 struct

    指定字段进行初始化：

    ```cpp
    #include <iostream>
    using namespace std;

    struct A
    {
        int val;
        int val_2;
    };

    int main()
    {
        A a = {
            .val = 1,
            .val_2 = 2
        };

        cout << a.val << endl;
        return 0;
    }
    ```
