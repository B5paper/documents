* [v] deal 函数

    ```matlab
    % 匿名函数配合 deal
    getTwoValues = @(x) deal(x^2, x^3);
    [a, b] = getTwoValues(3);  % a=9, b=27

    % 但注意：deal 在匿名函数中需要显式调用
    getMinMax = @(arr) deal(min(arr), max(arr));
    [minVal, maxVal] = getMinMax([5, 2, 8, 1]);
    ```

    deal 的限制：

    ```matlab
    % 这可以工作
    f = @(x) deal(x^2, x^3);
    [a, b] = f(2);  % a=4, b=8

    % 但这不行（匿名函数中不能有多个语句）
    % f = @(x) a = x^2; b = x^3; deal(a, b);  % 错误！
    ```
