* [ ] 调研下面这段 vim 自动跳转的代码

    **如果你想自动跳转，就必须自己加配置**

    把下面这段加入 `.vimrc` 或 `init.vim`：

    ```vim
    augroup restore_cursor
      autocmd!
      autocmd BufReadPost *
            \ if line("'\"") > 1 && line("'\"") <= line("$") |
            \   execute "normal! g`\"" |
            \ endif
    augroup END
    ```

    作用：

    * Vim 打开文件（BufReadPost）时
    * 如果 mark `"（上次退出点）在合法范围内
    * 自动执行 `g`"`（跳到那个位置）
